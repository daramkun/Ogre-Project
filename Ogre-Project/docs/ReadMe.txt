Ogre-Project Framework는 완전한 게임용 프레임워크 코드는 아니며
, 충남대학교 컴퓨터공학과 2011학년도 1학기 객체지향설계 팀 과제
인 'Ogre3D 게임 개발'을 위한 간소화된 게임 프레임워크입니다.

따라서 필요한 기능 외에는 구현하지 않았으며, 추가 기능의 사용을
 위해서는 추가 구현이 필요합니다. 현재 구현된 기능은 창 관리,
키 입력, 빛, 메쉬, 사운드, 상태 관리, 카메라, 타이머 등이며, 렌
더링은 Ogre3D를, 사운드는 ZPlay를 사용했습니다.

본 프레임워크에서 사용된 라이브러리인 Ogre3D는 MIT 라이선스를, 
ZPlay는 GNU 라이선스를 사용하고 있기 때문에 본 프레임워크는 두
라이선스를 모두 따릅니다. 단, 외부 공개를 목적으로 하고 있지 않
기 때문에 실제 적용 효과가 발휘되진 않을 것으로 생각됩니다.

현재 사용되는 프로젝트는 Visual Studio 2010용으로 만들어졌으며,
 OgreAppWizard를 이용하여 만들어진 프로젝트를 개량하여 사용하고
 있습니다. 또한 Release모드의 경우 멀티프로세서 컴파일을 기본으
로 사용하므로 컴파일 속도를 단축할 수 있을 것으로 생각합니다.
